## About

SWEETSTEVIAM is a professional website showcasing wholesale natural sweeteners including Stevia, Erythritol, Xylitol, Sucralose, Sorbitol, and other sugar alternatives.

## Features

- **SEO Optimized** - Clean URLs, sitemap.xml, Schema.org markup, meta tags, hreflang tags
- **Anti-Spam Protection** - Honeypot fields, timestamp validation, form expiration
- **Performance** - WebP images, optimized fonts, minified CSS, Gzip compression
- **Multilingual** - English (`/`) and Polish (`/pl/`) versions

## Tech Stack

- HTML5, CSS3, JavaScript/jQuery
- PHP with PHPMailer for contact forms
- Apache with mod_rewrite for clean URLs
